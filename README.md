# Plantify

A front-end plant store showcasing a responsive design and user-friendly interface.

**Features:**
- Home page with modern layout  
- Header with navigation  
- Countdown sale section  
- Store page with plant listings  
- Featured collections section  
- Contact form for user inquiries  
- Footer with essential links
